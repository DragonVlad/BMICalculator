# BMI Calculator

Это проект представляет собой простое приложение, которое помогает пользователям оценить свое состояние здоровья, рассчитывая индекс массы тела по формуле: вес (кг) / рост (м)^2.
Приложение предоставляет классификацию результата (underweight, healthy, overweight, obese).

## Основные возможности:
- Расчет ИМТ: принимает вес в килограммах и рост в метрах;
- Классификация результатов: определяет категорию веса на основе полученного ИМТ;
- Интуитивный интерфейс: простой и понятный пользовательский интерфейс.

## Использование:
1.  Введите ваш **вес** в соответствующее поле.
2.  Введите ваш **рост** в соответствующее поле (уточните единицы измерения, например, метры или сантиметры).
3.  Нажмите кнопку **"Calculate"**.
4.  Результат и его интерпретация отобразятся на экране.
